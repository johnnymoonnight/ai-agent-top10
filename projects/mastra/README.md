## 简介

Mastra 是现代 TypeScript 的 AI 应用和智能体框架。它提供了从早期原型到生产应用所需的一切。支持 40+ 模型提供商的 3000+ 模型、内置工作流引擎、MCP 服务器、可观测性和评估工具。可与 React、Next.js、Node.js 等集成。

## 核心功能

- **Model Router**: 40+ 提供商统一接口（3000+ 模型）
- **Agents**: 自主推理和工具调用
- **Workflows**: 图引擎工作流（.then()、.branch()、.parallel()）
- **Human-in-the-loop**: 暂停等待人工审批
- **MCP Server**: 编写 MCP 协议服务器
- **Observability**: 内置评估和可观测性
- **Context Management**: 对话历史、数据检索、记忆
