## 简介

MetaGPT 是一个多智能体框架，通过模拟软件公司的完整运作流程让多个智能体协作完成复杂任务。核心理念是"代码 = SOP(团队)"——将标准操作流程物化应用到 LLM 组成的团队中。

不同智能体扮演：
- **产品经理**: 撰写 PRD 文档
- **架构师**: 设计系统架构
- **项目经理**: 分配和追踪任务
- **工程师**: 编写代码实现
- **QA**: 测试和质量保证

## 核心功能

- 多角色协作智能体系统
- SOP 驱动的软件开发流水线
- 自然语言编程：一行需求 → 完整项目
- Data Interpreter：数据分析智能体
- MGX (MetaGPT X)：世界首个 AI 智能体开发团队产品

## 快速开始

```bash
# 安装
pip install --upgrade metagpt

# 初始化配置
metagpt --init-config

# 使用 CLI
metagpt "Create a 2048 game"
```
