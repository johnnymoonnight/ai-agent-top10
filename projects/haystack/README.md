## 简介

Haystack 是一个开源 AI 编排框架，用于构建生产级 LLM 应用。提供模块化管道和智能体工作流，对检索、路由、记忆和生成有精确控制。支持 RAG、多模态应用、语义搜索和自主智能体。被 Apple、Meta、Netflix、NVIDIA 等全球知名企业使用。

## 核心功能

- **Modular Pipelines**: 模块化 AI 管道设计
- **Agent Workflows**: 智能体工作流
- **Model-Agnostic**: 支持 OpenAI、Mistral、Anthropic 等多模型
- **Hayhooks**: REST API / MCP 服务器部署
- **Enterprise Platform**: 托管部署方案
- **Telemetry**: 匿名使用统计
